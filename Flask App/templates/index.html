Overview & Essential Question

  • The Goal of this project is to determine if a connection exists between home value and the home proximity to a grocery store.

  • Essential Question: What is the correlation between home value and proximity to Trader Joe's or Walmart?

  • The Stores:Trader Joe's and Walmart couldn't be more different as stores in terms of the demographics they target, the products they sell, the size of stores and even location.

  • Trader Joe's has built a brand as a supermarket selling niche food products that consumers can't find at other supermarkets. Their stores are moderately sized.

  • In contrast, Walmart is that one stop superstore that provides a large selection of low cost goods and is both a supermarket and general merchandise retailer. It has built a brand on affordability, where a family or individual’s dollar goes further.


The Data

  • Real Estate: Realtor.com dataset with property listings for the entire U.S. and Puerto Rico with price(Target Value), number of bedrooms and baths, house size, lot size and the address.


  • Walmart: Kaggle dataset with location information(address, zip code, city, state, latitude and longitude) for all U.S. Walmart stores.


  • Trader Joe's: Kaggle dataset with location information(address, zip code, city, state, latitude and longitude) for all U.S. Trader Joe's.

Model Features



What's the correlation Matrix telling us?





  • Reviewing the correlation of our features, we can see that Walmart and Trader Joe's proximities have correlation coefficients of -0.14 and 0.23, respectively. These correlations are not incredibly strong but the only feature in our model with a greater correlation is the number of beds at 0.29.


  • Despite these low values the direction of the correlations matched what we predicted. We were expecting property values to be negatively correlated with the Walmart proximity and positively correlated with proximity to Trader Joe's, and that’s indeed what we see.


  • Another feature worth noting from the correlation matrix is Acre Lot which is showing no correlation. Intuitively, this doesn't make sense, large multi-acre properties should be more valuable and we would expect to see a positive correlation. The little to no correlation we're seeing is likely due to the property types we examined in our model; more properties with little or no acres like apartments and condos.

    Conclusions

    • Correlation: No feature showed a strong correlation with home price, including Trader Joe’s and Walmart. However, the features that showed the strongest correlations with price can be seen with bath (0.29) and Trader Joe’s proximity (0.23). Walmart also has a reasonably strong correlation (-0.14), albeit negative.

    • Machine Learning: The extra tree regressor machine learning model proved to be the most effective model for the dataset. The R² value is 0.9293, meaning that the goodness of fit of the regression model is 93%.




    • Web Application: By using JavaScript and HTML, our group was able to launch a web application with multiple user interactive elements, allowing a user to accurately predict a house value based on the selected features.

    • Going Further: A further investigation into the impact of using distance from Trader Joe’s and Walmart (given latitude and longitude) could impact correlation and still needs to be studied. Another next step could include increasing functionality of the website to allow end users to type an address and get a prediction of a home value.
